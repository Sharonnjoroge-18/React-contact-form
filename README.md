# React Contact Form

## Overview
This project is a contact form built using React, designed to demonstrate form handling, validation, and user interaction in a modern front-end application.

## Features
- Form input handling using React state
- Validation for required fields
- Error handling with user feedback
- Form submission handling
- Styled user interface using CSS

## Purpose
The goal of this project was to practice:
- Managing form state in React
- Implementing validation logic
- Handling user input and submission events
- Improving UI with CSS styling

## Technologies Used
- React
- JavaScript
- CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open in browser at `http://localhost:3000`
